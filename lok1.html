Type R

Honda Civic Type R er afkastamikil útgáfa af Civic Honda fyrirferðarlítill bíll. Type R merkið var fyrst notað á japönskum innanlandsmarkaði JDM Civic árið 1997 og það varð fljótt þekkt fyrir afkastagetu og brautarhæfni. Þó að hún sé ekki fyrsta Honda módelið til að hljóta Type R meðferðina og hið táknræna rauða merki er hún sú vinsælasta og hefur haldið áfram í gegnum mismunandi kynslóðir Civic.
